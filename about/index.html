About Me

I am Simon Brebeck - a self-taught full-stack developer with passion to create all kinds of appealing software. Currently, I live in the beautiful city of Aachen, Germany, studying Computer Science at the RWTH Aachen University.

When I was 11 years old, I started programming extensions for a video game called Minecraft with the help of youtube videos. I loved creating my own ideas and seeing them come to life. This is what sparked my interest in programming.

But at some point, I wanted to create more than just small Minecraft plugins with Java and MySQL. So I began to learn Sockets to create a Chat Application that I could present as a project in school. This was the first time I programmed something in plain Java without any frameworks or libraries.

After that, I started to learn more about web development and created my first website with HTML, CSS and JavaScript. I was amazed by how easy it was to create a website and how much I could do with it.

Nowadays, I am working on my own projects and learning new technologies like Spring Boot. See my skills below.
