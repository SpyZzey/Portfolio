Calendar

Calendar is a web-based calendar application that allows users to create, edit, and delete events. The application is built with HTML, CSS, JavaScript, PHP, jQuery, and MySQL. It is hosted on a virtual private server with Apache2.

The application is built with a mobile-first approach and is fully responsive. It is designed to be used on mobile devices, but can also be used on desktop computers.

The design is inspired by Samsung's Calendar app.

Features

  • Dark Mode
    Calendar supports a dark mode for users who prefer a darker theme.
  • Multiple languages
    Calendar supports multiple languages. Currently, English and German are supported
  • Responsive Design
    Calendar is designed to be used on any device.
  • Event Creation, Editing and Deletion
    Calendar allows users to create, edit and delete events.
