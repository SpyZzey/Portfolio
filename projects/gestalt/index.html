quirium

Early Stage!
A web-based product management and development design tool.

    A tool that is designed to support the development process of apps. Create database designs, export database sql scripts, create wireframes, manage tasks and more...

    Implemented with Angular and Supabase, designed to be used by teams in a collaborative way.

    Based on previous work: Database Designer

Features

  • Realtime Collaboration
    Work in realtime with other team members on your project.
  • Database Designer
    Design your database and automatically parse it to sql, cds, ...
  • Kanban Board
    Manage your planned features in a kanban board.
  • Wireframing
    Plan and design the layout of your app.
